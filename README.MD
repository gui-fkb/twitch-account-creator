## 💻 Installation

#### You can [download the project for Windows/Linux](https://github.com/masterking32/twitch-account-creator/releases) or Run in it manually.

### 🟣 Setup:
- Clone the repo and do `npm install`

### 🟣 Account creator:
- Put your `YOUR Solver API KEY` API key in `config.js`
- Put your `Proxies` API key in `config.js`
- Run the project (`node main.js`).
